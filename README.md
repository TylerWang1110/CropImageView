# CropImageView
## 可自由指定自适应裁剪对齐点的ImageView

Add it in your root build.gradle at the end of repositories:

	allprojects {
		repositories {
			...
			maven { url 'https://www.jitpack.io' }
		}
	}
Step 2. Add the dependency

	dependencies {
	        implementation 'com.github.goweii:CropImageView:v1.0.1'
	}
